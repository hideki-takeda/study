制御構造

代表的な制御は以下の3つ

  • 順次
  • 分岐
  • 繰り返し

if(条件){
条件がYesの時の処理
}else{
Yesじゃない（そうでない）時の処理
}
(条件 == 〇〇)
例）
if(num%2 == 0)：偶数ならば
if(num%2 == 1)：奇数ならば


boolean tenki = true; if(tenki == true){
system.out.println("選択をします")
}else{
system.out.println("DVDを見ます")
}

while(条件){
条件がYesの時の処理
}
(条件 == 〇〇)

条件が成立している限り、処理を続ける


分岐の構文
if(条件式){
ブロック
}else{
ブロック
}

繰り返しの構文
while(条件式){
ブロック
}

変数が利用可能な場所の範囲のことをスコープとよぶ

関係演算子
==　左辺等辺が等しい
!=　左辺等辺が異なる
など、、

if文やwhile文では評価がtrue / false で処理を決める
(条件式)の評価は、評価結果がtrueかfalseになる必要がある

※注意
文字列型の変数.equals(比較相手の文字列)
→文字列は「==」で比較できない

例）
×if(str == "夕日"){

〇if(str.squals("夕日")){

論理演算子
&& かつ　（両方の条件が満たされたときにtrue）
|| または　（どちらか片方の条件が満たされればtrue）
