制御構造

代表的な制御は以下の3つ

  • 順次
  • 分岐
  • 繰り返し

if(条件){
条件がYesの時の処理
}else{
Yesじゃない（そうでない）時の処理
}
(条件 == 〇〇)
例）
if(num%2 == 0)：偶数ならば
if(num%2 == 1)：奇数ならば


boolean tenki = true; if(tenki == true){
system.out.println("選択をします")
}else{
system.out.println("DVDを見ます")
}

while(条件){
条件がYesの時の処理
}
(条件 == 〇〇)

条件が成立している限り、処理を続ける
